# Landing Page - Miolo Comunicação Gráfica

Este projeto consiste em uma Landing Page institucional desenvolvida para a gráfica Miolo Comunicação Gráfica, com o objetivo de apresentar serviços, materiais gráficos e facilitar o contato com clientes. A aplicação utiliza HTML, CSS e Java, além de uma API REST responsável por processar informações enviadas através do formulário de contato.

# Tecnologias Utilizadas

•HTML5 — Estrutura da página

•CSS3 — Estilização e layout responsivo

•Java — Desenvolvimento do backend

•API REST — Comunicação entre frontend e backend

•Git / GitHub — Controle de versão

# Objetivo do Projeto

O objetivo deste projeto é criar uma landing page moderna e responsiva para apresentar os serviços da gráfica, como:

•Banners

•Panfletos

•Lonas

•Materiais personalizados

Além disso, o projeto inclui uma integração com API REST, permitindo o envio de dados de contato para o servidor.

# Funcionalidades

•Página institucional da gráfica

•Apresentação de materiais gráficos

•Layout responsivo

•Seção de contato

•Integração com API REST em Java
